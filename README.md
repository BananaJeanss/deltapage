A collection of Deltarune/Undertale media and tools.

## Description

Features pages such as a art gallery, textbox generator, radio (using tsradio as backend), dialogue/sprites/maps list/view (data extracted via undertalemodtool and then organized/sorted), and some other stuff.

## Quick Start (contributing)

1. Clone the repository

    ```bash
    git clone https://github.com/BananaJeanss/deltapage.git
    cd deltapage
    ```

2. Install dependencies

    ```bash
    npm i
    ```

3. Setup .env using .env.example (make sure to change the values)

    ```bash
    cp .env.example .env
    ```

4. Start development server

    ```bash
    npm run dev
    ```

5. Open your browser and navigate to `http://localhost:3000`
